# 🌍 Wanderlust — Travel Dashboard

Wanderlust is a modern travel dashboard built using **HTML, CSS, JavaScript**, and multiple external APIs.
It helps users explore countries, public holidays, events, weather, currencies, and travel information through a clean, responsive, and interactive interface.

The project focuses on API integration, asynchronous JavaScript, modular architecture, and dynamic UI updates.

## 📌 Features

✅ Explore countries with detailed information  
✅ Search countries using a custom searchable dropdown  
✅ View public holidays by country and year  
✅ Discover upcoming events and attractions  
✅ Weather forecasts for selected cities  
✅ Currency exchange rates with live conversion  
✅ Sunrise & Sunset information  
✅ Long weekend finder for trip planning  
✅ Save favorite holidays and events using Local Storage  
✅ Responsive design for desktop, tablet, and mobile  
✅ Clean and modern travel-inspired UI

---

## 🛠 Tech Stack

- **HTML5** — Structure and semantic markup
- **CSS3** — Styling, animations, and responsive layout
- **JavaScript (ES6 Modules)** — Dynamic functionality and API integration
- **REST APIs** — Fetching live travel data
- **Local Storage** — Saving user plans
- **SweetAlert2** — Beautiful alerts and notifications
